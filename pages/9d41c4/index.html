深浅拷贝

# 赋值

不会在堆中开辟新空间，赋的值是内存地址，共享同一块内存，值类型和引用类型都相互影响

const a = {
  name: 'name',
  type: {
    age: 12
  }
}
const b = a
// name、type、属性都会相互影响

# 浅拷贝

在堆中开辟新空间，第一层的属性值类型互不影响，引用类型相互影响

const a = {
  name: 'name',
  type: {
    age: 12
  }
}
const b = a
// name不会项目影响
// type、age会相互影响
